 3 - De acordo com a estrutura abaixo:
      typedef struct{
        int matricula;
        float nota1, nota2;
      }alunos;

4 - Um programa que crie uma matriz 10x10, com números aleatórios de -20 até 20 incluindo-os, imprima a matriz.
Crie uma função que receba a matriz e retorne a soma de todos os numeors abaixo da diagonal principal (incluindo a diagonal).
O programa principal imprime o reesultado da função.
